# Installation

* Please install nodejs before install this script
```shell

> git clone https://github.com/Kynders/Kynderbot
> cd Kynderbot
> npm install
```
# How to use

* Default
```bash
> npm start
```

* If using QR-Code
```bash
> npm run code
```

* If using Pairing Code
```bash
> npm run main
```
